# Format: nama|email|phone|subdistrik_search|alamat_detail|kecamatan_note|kode_pos_note|size|qty|fallback
#
# Kolom 1 : nama
# Kolom 2 : email
# Kolom 3 : phone (08xxx / +62xxx)
# Kolom 4 : kata kunci sub-district dropdown (misal: "Tanah Abang Jakarta Pusat")
# Kolom 5 : alamat detail (field "Address Details")
# Kolom 6 : note District/Kecamatan
# Kolom 7 : note Postal Code/Kode Pos
# Kolom 8 : size/color variant (bisa list prioritas dipisah koma)
#            - "M"              -> cari M
#            - "M,S"            -> cari M dulu, kalau habis pindah S
#            - "BLACK,PINK"     -> cari BLACK dulu, kalau habis PINK
#            - "M,S,RANDOM"     -> M, lalu S, lalu random dari yg ready
#            - kosong           -> ambil yg ready (produk tanpa variant otomatis)
# Kolom 9 : qty / jumlah beli (default: 1)
# Kolom 10: fallback mode kalau semua preferensi habis
#            - "wait" (default)   -> tunggu retry 0.1s sampai variant muncul
#            - "random"           -> ambil random dari yg ready
#
